# End-to-End DevSecOps Platform

## Project Description
This project implements a complete DevSecOps platform for a cloud-native web application.
It covers CI/CD automation, containerization, Kubernetes orchestration, security integration,
monitoring, and infrastructure as code.

## Technologies
- Git & GitHub
- Docker & Docker Compose
- GitHub Actions
- Kubernetes
- Terraform & Ansible
- Prometheus & Grafana
- SonarQube & Trivy
